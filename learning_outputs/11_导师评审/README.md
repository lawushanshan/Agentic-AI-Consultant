# 导师评审目录

这个目录用于保存 AI 学习导师对阶段成果的评审记录。

建议每次完成一个关键学习产出后，都生成一份评审记录。

---

## 1. 什么时候调用导师

建议在这些节点调用：

- 第 1 周：Prompt 练习集完成 10 个案例后
- 第 4 周：LLM API / Web API 阶段复盘
- 第 8 周：RAG 项目完成后
- 第 12 周：工具调用 Agent 完成后
- 第 18 周：PRD、架构、测试集、治理和上线计划完成后
- 第 24 周：作品集和专业交付包完成后

---

## 2. 推荐指令

```text
请调用 AI 学习导师，评审我第 1 周的 Prompt 练习集，并生成评审记录。
```

```text
请调用 AI 学习导师，按第 8 周标准评审我的 RAG 原型说明。
```

```text
请调用 AI 学习导师，模拟面试官追问我的专业项目交付包。
```

## 4. 使用原则

评审记录不是为了证明自己已经完成，而是为了暴露问题并推动修改。

一份合格的评审记录应该包含：

- 明确结论
- 达标项
- 主要问题
- 必须修改项
- 建议优化项
- 下一步行动
