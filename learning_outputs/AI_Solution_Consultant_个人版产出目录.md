# AI Solution Consultant 个人版产出目录

适用对象：ASP.NET Core 程序员，正在向技术型 AI Solution Consultant / Agentic AI Delivery Consultant 转型。

使用原则：

- 原 `_模板.md` 文件保留不动。
- 所有 `_个人版.md` 文件用于记录自己的真实学习和项目成果。
- 每周至少更新一个个人版文件。
- 每个项目都要同时体现技术、业务、风险和交付能力。

## 3. 推荐项目主线

第一条项目主线：

> 企业知识库问答 Agent

用于训练 RAG、引用来源、无答案拒答、文档权限和测试集。

第二条项目主线：

> 客服工单分类 Agent

用于训练结构化输出、工具调用、人工确认、日志、风险控制和评估。

第三条项目主线：

> 销售续约分析 Agent 或采购申请初审 Agent

用于训练更接近真实企业系统的多工具调用、权限模型、审批和 ROI。

## 4. 每周更新要求

每周更新时至少完成：

- 一个新增章节
- 一组测试数据
- 一个失败案例
- 一个改进动作

不要只写“已完成”，要留下可复查的证据。
