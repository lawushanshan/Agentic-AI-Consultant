# Agentic AI Consultant 学习文档使用指南

版本日期：2026-06-11

这个文件夹下的文档不是并列阅读材料，而是一套分层学习体系。

---

## 1. 三份核心文档怎么用

| 文档 | 作用 | 适合谁 | 使用方式 |
|---|---|---|---|
| `Agentic_AI_Consultant_就职与学习手册.md` | 职业总纲 | 所有人 | 先快速读一遍，建立全局地图 |
| `Agentic_AI_Consultant_零基础学习路线.md` | 入门训练 | 非计算机新人、刚毕业学生 | 按 12 周执行，每周交付一个成果 |
| `Agentic_AI_Consultant_专业交付教程.md` | 项目交付手册 | 技术、产品、咨询、实施、RPA、数据人员 | 按项目生命周期使用，训练真实交付能力 |
| `AI_Solution_Consultant_个人转型教程.md` | 个人转型路线 | 有程序员背景、想转 AI Solution Consultant 的人 | 按 24 周训练，形成作品集和专业交付包 |
| `AI_Solution_Consultant_24周学习日程.md` | 周计划执行表 | 已确定转型方向、需要按周推进的人 | 从 2026-06-15 开始，每周明确任务和交付物 |
| `AI学习导师_角色说明.md` | 导师评审规则 | 需要阶段评审、自学纠偏、面试追问的人 | 完成阶段产出后调用导师进行 review |

推荐顺序：

```text
先读职业总纲
  -> 判断自己属于新人还是专业人员
  -> 新人走零基础路线
  -> 专业人员走专业交付教程
  -> 最后用项目作品集验证学习成果
```

## 2. 新人怎么学

新人不要从专业交付教程开始。

正确路线：

1. 用 1-2 天读职业总纲，理解岗位是什么
2. 按零基础路线完成 12 周学习
3. 每周必须有一个可检查产出
4. 第 12 周整理成入门作品集
5. 再进入专业交付教程

新人最低合格标准：

- 能解释 Agent、RAG、工具调用、人工审批
- 能写 20 个基础 Prompt 案例
- 能画 1 张业务流程图
- 能做 1 个知识库问答原型
- 能设计 1 组工具卡片
- 能完成 1 个小型 Agent 项目说明
- 能讲清项目的价值和风险

---

## 3. 专业人员怎么学

专业人员不要只读概念。

正确路线：

1. 选择一个真实或模拟业务场景
2. 按专业交付教程完成从机会识别到上线计划的完整交付包
3. 至少做 20 条测试用例
4. 至少设计 3 个工具或 1 个 RAG 方案
5. 补齐权限、审批、日志、评估和回滚方案

专业人员最低合格标准：

- 能输出场景机会清单
- 能画 As-Is / To-Be 流程图
- 能写 Agent PRD
- 能设计 RAG 或工具调用方案
- 能设计测试集和评估指标
- 能设计权限、审批和审计方案
- 能写上线计划和运营报告模板

## 4. 学习成果如何验收

不要用“看完了”作为学习完成标准。

用产出验收：

| 阶段 | 验收产出 |
|---|---|
| 职业认知 | 职业理解一页纸 |
| Prompt | 20 个 Prompt 案例 |
| 业务流程 | 1 张流程图和痛点说明 |
| RAG | 1 个知识库问答原型说明 |
| 工具调用 | 5 张工具设计卡片 |
| 风险控制 | 1 张风险清单 |
| 综合项目 | 1 份 Agent 项目说明 |
| 专业交付 | 1 个完整项目交付包 |

一个学习者能拿出这些产出，才说明文档真正被用起来了。

建议在关键节点调用 `AI学习导师_角色说明.md` 中定义的导师角色做评审：

```text
请调用 AI 学习导师，评审我第 1 周的 Prompt 练习集，并生成评审记录。
```

评审记录建议保存到：

```text
learning_outputs/11_导师评审/
```

## 6. 教学者怎么带学员

如果你要用这套文档培训新人，建议每周检查 3 件事：

1. 学员是否能用自己的话解释本周概念
2. 学员是否完成了本周产出
3. 学员是否记录了失败案例和改进方法

不要只讲课。每周都要让学员提交一个小成果。

如果没有真实教学者，可以让 AI 学习导师承担阶段评审角色。导师不负责陪读，而是负责检查产出是否达标、指出问题、追问业务价值和风险边界。

## 7. 最重要的使用原则

这套文档的核心不是阅读，而是训练。

真正的学习路径是：

```text
读懂一个概念
  -> 做一个小练习
  -> 记录一次失败
  -> 改进一次方案
  -> 沉淀一个作品集材料
```

只要坚持这个闭环，零基础新人可以逐步入门，专业人员也可以把已有经验升级为 Agentic AI 项目交付能力。
